# Seabound

A multiplayer co-op naval combat and exploration game developed with Unity and Netcode for GameObjects (NGO). 

## Features
- **Multiplayer:** Full Host/Client support using NGO.
- **Combat:** Drive boats, engage enemy ships, and fight AI guards using hitscan and spread-based weapons.
- **Economy:** Defeat enemies, steal treasure chests, and sell them to the merchant.
- **Systems:** Fully functioning inventory, shop, and a persistent JSON save system.

## Controls
- **W,A,S,D:** Move / Drive
- **Space:** Jump
- **Right Click:** Aim
- **Left Click:** Shoot
- **F:** Interact / Board / Pick up / Sell
- **1, 2:** Switch Weapons
- **3:** Use Bandage
- **Tab:** Inventory
- **F1:** Debug Menu

## Setup
1. Open in Unity 6.3.15
2. Open the `Main` scene.
3. Build for Windows. Run one instance as Host and another as Client to test multiplayer.
